# Zypi Space — Política de Privacidade

Página estática da **Política de Privacidade** da Zypi Space, em conformidade com a
Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), publicada via GitHub Pages.

## Estrutura
- `index.html` — a política de privacidade (página única, responsiva, autocontida).
- `assets/logo.svg` — ícone/logo (favicon e usos auxiliares).
- `assets/cover.svg` — imagem de capa (Open Graph / compartilhamento).
- `.nojekyll` — garante que o GitHub Pages sirva os arquivos sem processamento Jekyll.

## Publicação
Publicado automaticamente pelo GitHub Pages a partir da branch `main` (pasta raiz).

## Pendências de conteúdo
Há marcadores `[preencher]` / `[confirmar]` no `index.html` para dados legais que
devem ser revisados pelo responsável: razão social, CNPJ, e-mail de contato/DPO e WhatsApp.

> Para usar as artes originais em PNG no lugar dos SVGs, adicione `assets/logo.png`
> e `assets/cover.png` e atualize as referências no `index.html`.
